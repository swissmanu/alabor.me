TalkBeamer

Use TalkBeamer to present the slides you created using the LaTeX Beamer template from your iPad or iPhone.

  • Show slides on an external connected projector or display (wired or through AirPlay screen mirroring)
  • Display notes for the current slide on your iPad or iPhone
  • Navigate between slides with ease
  • Keep track of your presentation time using the presentation timer
  • Open a PDF from any place through the iOS sharing capability
  • Present PDF slides without notes: Your iPad or iPhone will show the current slide

TalkBeamer Pro

TalkBeamer for lightning talks is free. This means you can do presentations up to 8 minutes before TalkBeamer stops displaying slides on the external display.

If you like this app and want to use it for presentations longer than 8 minutes, consider a one-time purchase of TalkBeamer Pro.

What is LaTeX? What is LaTex Beamer?

LaTeX is a software package to author various documents like scientific reports, magazine articles or even books.

In combination with the Beamer template, LaTeX's powerful feature set can be used for the creation of slide sets for talks and presentations. When included, exported PDF files contain the actual slide as well as notes for helping the speaker.
